## 🌟 Overview

**FINGPT-21 AI** is an intelligent financial analysis platform that combines real-time stock market data with advanced AI capabilities. The application enables users to analyze stock performance, explore technical indicators, visualize market trends, and interact with an AI-powered chatbot for financial insights.

Built using **Streamlit**, **Yahoo Finance**, **OpenAI**, and **Google Gemini**, the platform delivers an intuitive and data-driven experience for investors, students, researchers, and finance enthusiasts.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/tharunprinz/FinGPT-21.git
cd FinGPT-21
```

### Create Virtual Environment

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

#### macOS/Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file in the root directory:

```env
OPENAI_API_KEY=your_openai_api_key
GEMINI_API_KEY=your_gemini_api_key
```

---

## ▶️ Run the Application

```bash
streamlit run app/main.py
```

Application will be available at:

```bash
http://localhost:8501
```
